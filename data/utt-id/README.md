# utt-id 目录说明

本目录已按你要求重排为两级结构：

- `utt-id/train/lite_1h`
- `utt-id/train/lite_30h`
- `utt-id/test/lite_1h`
- `utt-id/test/lite_30h`

## 低资源语种配置差异
- **1h 配置**：低资源语种（如 ar/ja/ko/pt）使用 1 小时级别训练配置（按 Table VI）。
- **30h 配置**：
  - Arabic / Japanese / Korean 来自 Common Voice（其中 Korean 官方可用时长约 6.51h）。
  - Portuguese 来自 MLS。
